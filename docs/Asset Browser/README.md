# Asset Browser

Although distributed not under my name, I still am the creator and maintainer.

This tool allows you to create a user-friendly webpage with all the files from your github assets repository shown and links attached.

# Usage:

1. Replace BASE_URL with your path to the files
2. Replace "GITHUB_REPO" & "GITHUB_USERNAME" with yours
    - (note: those are not env. values as the code is completed on client side)

That's it!
